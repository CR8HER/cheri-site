C) JavaScript - top

This Html imports four Javascript files.

  1. jQuery
  2. My custom scripts js file js/custom.js
  3. Bootstrap js file js/bootstrap.min.js
  4. Some plugin owlcarousel, bootstrap-datetimepicker
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Most of the animation in this site is carried out from the customs scripts. There are a few functions worth looking over. 
    	// JavaScript Document

    /*--
		    Header Sticky

        var scroll = $(window).scrollTop();

            $(".header-bottom, .header-mobile").addClass("sticky");

F) CSS Files and Structure - top

I'm using two CSS files in this Html. The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.
